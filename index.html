O que é devPlay.app?

O web app devPlay foi desenvolvido principalmente para ajudar quem está iniciando nos estudos do desenvolvimento front-end ou simplesmente desenvolvimento web, como queira chamar. O foco é nas três tecnologias básicas e fundamentais para o desenvolvimento de sites e aplicações web, que são HTML, CSS e JavaScript. Basicamente seu objetivo é editar e ver snippets ou blocos de códigos acompanhando sua execução ao vivo, usando apenas o próprio navegador de internet.

Também pode ser usado por desenvolvedores experientes pra criar uma "biblioteca" de snippets de códigos com o objetivo de servir como referência ou mesmo para reutilização em projetos maiores, ganhando assim muito mais tempo para gastar na parte criativa do seu projeto.

Salve seus projetos

O devPlay salva seus projetos no armazenamento local do próprio navegador. Isto significa que qualquer dispositivo que você usar, computador ou smarthpone, ou mesmo que use o mesmo dispositivo com sessões/login diferentes, cada um deles terá seu próprio armazenamento independente ao usar o devPlay.

Como usar?

Clique no botão "Projects" para criar ou acessar seus projetos. Para criar um novo projeto digite um nome no campo "New Project" e clique em "OK". Todos os projetos criados ficam alistados na caixa de seleção acima "Projects". Selecione o projeto que deseja editar ou visualizar e clique em "OK".

Agora é só codar! Simples assim. Ao digitar código HTML e CSS nas caixas correspondentes, automaticamente o devPlay salva seu código no navegador e atualiza a previsualização do seu projeto. O código JavaScript não é salvo automaticamente para evitar problemas causados por execução não desejada, enquanto você escreve as funções do seu programa. Clique no botão "JS" para salvar seu código, ou simplesmente altere algo nas caixas de HTML ou CSS. O código será salvo e executado.

Previsualização e visualização em tela cheia

A qualquer momento você pode desabilitar ou habilitar a visualização prévia do seu projeto, clicando no botão "Preview". Para visualizar o projeto em tela cheia numa nova guia do navegador clique no botão "Window". Se quiser trabalhar desse forma usando a visualização numa nova janela, basta recarregar a página de visualização sempre que quiser ver as suas últimas alterações.

Layout personalizado para cada dispositivo

A interface de devPlay foi desenvolvida para tentar aproveitar ao máximo o tamanho da tela de cada dispositivo. Por isso o layout para dispositivo móvel é diferente do layout para desktop. No celular você pode alteranar facilmente entre as caixas de código clicando nos botões correspondentes.

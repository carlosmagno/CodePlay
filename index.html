O que é HTML?

HTML é a sigla de HyperText Markup Language, em português "Linguagem de Marcação de Hipertexto". Dito de modo simples é a linguagem responsável pelo conteúdo das páginas web. Faz parte das três principais tecnologias responsáveis por tudo o que vemos na internet. São elas com suas respectivas responsabilidades:

  1. HTML - conteúdo (como este texto, imagens etc...)
  2. CSS - aparência/ apresentação
  3. JavaScript - funcionalidade/ interação

Essas linguagens, assim como muitas outras, são interpretadas pelos navegadores de internet como o Internet Explorer, Edge, Google Chrome e outros e possibilitam a grande facilidade de comunicação e compartilhamento de conteúdo na rede mundial. Mas o nosso objetivo neste curso é ensinar para você HTML 5. O número "5" se refere a versão atual desta linguagem, no momento da produção deste curso obviamente.

Algo muito importante a se mencionar já agora é que como o próprio nome da linguagem diz, ela é uma linguagem de "marcação". Portanto não é correto dizer que você "programa" em HTML. Alguém pode programar em JavaScript por exemplo, mas não em HTML, embora as duas tecnologias possam geralmente estar relacionadas (no desenvolvimento front-end principalmente). Então para não ser "mal visto" por programadores ou até um possível contratante, evite esta "gafe". Você pode dizer que está aprendendo a desenvolver em HTML, já que o HTML é a linguagem mais básica para o desenvolvimento WEB.

Você escreve HTML usando "elementos". Elementos são palavras específicas reservadas envolvidas em "tags" "<" e "/>". Esses elementos definem o tipo de conteúdo que será incluído dentro deles. Geralmente os elementos tem tags de abertura e fechamento. As tags de fechamento tem a barra "/" antes do nome do elemento. HTML é uma linguagem "case insensitive", ou seja é insensível a caixa das letras. Então tanto faz escrever os nomes dos elementos e letras maiúsculas ou minúsculas ou uma combinação de ambos. O navegador vai interpretar do mesmo jeito.

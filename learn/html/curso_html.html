1.1 Afinal, o que é HTML?

HTML é a sigla de HyperText Markup Language, em português "Linguagem de Marcação de Hipertexto". Dito de modo simples é a linguagem responsável pelo conteúdo das páginas web. Faz parte das três principais tecnologias responsáveis por tudo o que vemos na internet. São elas com suas respectivas responsabilidades:

  1. HTML - conteúdo (como este texto, imagens etc...)
  2. CSS - aparência/ apresentação
  3. JavaScript - funcionalidade/ interação

Essas linguagens, assim como muitas outras, são interpretadas pelos navegadores de internet como o Internet Explorer, Edge, Google Chrome e outros e possibilitam a grande facilidade de comunicação e compartilhamento de conteúdo na rede mundial. Mas o nosso objetivo neste curso é ensinar para você HTML 5. O número "5" se refere a versão atual desta linguagem, no momento da produção deste curso obviamente.

Algo muito importante a se mencionar já agora é que como o próprio nome da linguagem diz, ela é uma linguagem de "marcação". Portanto não é correto dizer que você "programa" em HTML. Alguém pode programar em JavaScript por exemplo, mas não em HTML, embora as duas tecnologias possam geralmente estar relacionadas (no desenvolvimento front-end principalmente). Então para não ser "mal visto" por programadores ou até um possível contratante, evite esta "gafe". Você pode dizer que está aprendendo a desenvolver em HTML, já que o HTML é a linguagem mais básica para o desenvolvimento WEB.

Você escreve HTML usando "elementos". Elementos são palavras específicas reservadas envolvidas em "tags" "<" e ">". Esses elementos definem o tipo de conteúdo que será incluído dentro deles. Geralmente os elementos tem tags de abertura e fechamento. As tags de fechamento tem a barra "/" antes do nome do elemento. 1.2 Estrutura básica de um documento HTML

Na parte superior do documento, é necessário informar ao navegador qual versão de HTML sua página está usando. HTML é uma linguagem em evolução e é atualizada regularmente. A maioria dos principais navegadores suporta a especificação mais recente, que é HTML5. No entanto, as páginas da web mais antigas podem usar versões anteriores do idioma.

Você fornece essas informações ao navegador adicionando a tag <!DOCTYPE...> na primeira linha, onde a parte "..." é a versão do HTML. Para HTML5, você usa <!DOCTYPE html>.

<html> </html>

Dito de modo simples é a raiz ou o elemento pai de todos os outros elementos de um documento HTML. É dentro desse elemento que toda a estrutura principal do documento é construída.

<head> <head>

Em português "head" é traduzido "cabeça". Esse é o primeiro elemento filho dentro das tags HTML. No head são colocadas as principais informações sobre a página, essenciais para os navegadores de internet. Por exemplo o título da página, o tipo do conteúdo, configurações, links para documentos externos etc... As maioria dessas informações não são visíveis ao leitor da sua página mas são essencias para o navegador exibir corretamente seu site como veremos mais adiante.

<body> </body>

Como o próprio nome do elemento diz trata-se do "corpo" do documento. Ou seja é aqui que você vai escrever o conteúdo propriamente dito da sua página. É o segundo elemento filho dentro das tags HTML.

1.3 Criando sua primeira página HTML

Com as informações que já temos até aqui podemos começar a construir nosso primeiro documento HTML ou seja nosso site. Claro que vai ser um site supersimples, mas a idéia agora é você aprender os conceitos principais. Vamos usar o nosso editor online.

Crie um novo projeto no DevPlay e dê o nome que preferir. Sei lá, pode ser algo como "Meu Primeiro Projeto HTML". Na área do código HTML escreva o seguinte:

<title> </title>

Note que já estamos usando outro elemento html dentro do head do nosso site, que é o elemento title. Essa tag diz ao navegador qual é o título da página. O título é o que aparece quando você descansa o mouse em cima da guia do navegador para cada site aberto. No nosso caso o título é "Meu Primeiro Projeto HTML".

1.4 Informações do site

Agora vamos adicionar informações importantes sobre o nosso site que vão ajudar o navegador a saber exatamente a forma que deve se comportar ao abrir o nosso site. Todas essas informações são inseridas no head do documento html dentro da tag meta.

<meta> </meta>

A tag meta vem de metadados que em termos simples se refere a informação sobre a informação. Cada tag meta tem basicamente um atributo "name" com o nome da informação que se deseja passar para o navegador e um atributo "content" com o contéudo daquela informação. No exemplo a seguir estamos informando:

  • O tipo de codificação de texto a ser usado pra renderizar a página (UTF-8)
  • Instruções sobre a largura da viewport (área de visualização do navegador)
  • O nome do autor da página
  • A descrição resumida sobre o conteúdo da página

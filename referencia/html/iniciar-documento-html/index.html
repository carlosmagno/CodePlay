Olá mundo! Eu sou um parágrafo.

Na parte superior do documento, é necessário informar ao navegador qual versão de HTML sua página está usando. HTML é uma linguagem em evolução e é atualizada regularmente. A maioria dos principais navegadores suporta a especificação mais recente, que é HTML5. No entanto, as páginas da web mais antigas podem usar versões anteriores do idioma.

Você fornece essas informações ao navegador adicionando a tag <!DOCTYPE...> na primeira linha, onde a parte "..." é a versão do HTML. Para HTML5, você usa <!DOCTYPE html>.

<html> </html>

Dito de modo simples é a raiz ou o elemento pai de todos os outros elementos de um documento HTML. É dentro desse elemento que toda a estrutura principal do documento é construída.

<head> <head>

Em português "head" é traduzido "cabeça". Esse é o primeiro elemento filho dentro das tags HTML. No head são colocadas as principais informações sobre a página, essenciais para os navegadores de internet. Por exemplo o título da página, o tipo do conteúdo, configurações, links para documentos externos etc... As maioria dessas informações não são visíveis ao leitor da sua página mas são essencias para o navegador exibir corretamente seu site como veremos mais adiante.

<title> </title>

Note que já estamos usando outro elemento html dentro do head do nosso site, que é o elemento title. Essa tag diz ao navegador qual é o título da página. O título é o que aparece quando você descansa o mouse em cima da guia do navegador para cada site aberto. No nosso caso o título é "Meu Primeiro Projeto HTML".

<body> </body>

Como o próprio nome do elemento diz trata-se do "corpo" do documento. Ou seja é aqui que você vai escrever o conteúdo propriamente dito da sua página. É o segundo elemento filho dentro das tags HTML.
